Logiciel Perlea

Perlea est un logiciel qui permet de créer facilement ses grilles de perles à partir d'une image. Pour créer une grille, aller dans la barre de menu : Créer une grille. Choisir le modèle que vous voulez et changer les options disponibles.

Conseil pour une grille réussie :

  • Les images avec des couleurs bien différenciées auront un meilleur rendu. La grille contient 4, 8 ou 16 couleurs. Il y aura donc forcément une perte du détail sur des photos contenant plusieurs milliers de couleurs par exemple.
  • Le programme redimensionnant les images avant la création de la grille, les petites images contenant peu de détails auront un meilleur rendu aussi.

Perlea is a software who creates beadweaving grids from an image. To create one, go to menu bar : Create a grid. Choose type of grid you want and change available options.

Tips & tricks for a good grid :

  • Images with different colors will give a best result. A grid are in 4, 8 or 16 colors. So there are a leak of detail on photographies of thousand of colors.
  • The program resizes images before the creation of the grid. So no-detailed images will give a best result.
